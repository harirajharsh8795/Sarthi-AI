# Saarthi AI Knowledge Base

Welcome to the Saarthi AI Knowledge Base. This repository contains comprehensive, multilingual (English, Hindi, Hinglish) information designed for the Saarthi AI assistant to provide accurate, helpful, and localized responses.

## Structure
The knowledge base is organized into major domains:
- **Banking:** Covers banking basics, accounts, cards, loans, investments, cyber security, government schemes, RBI policies, and payments.
- **Legal:** Covers the Indian constitution, criminal law (BNS), family law, property law, consumer law, labour law, and police procedures.
- **Medical:** Covers anatomy, chronic diseases, infectious diseases, child health, women's health, diagnostic tests, first aid, medicines, and symptoms.
- **Common:** Contains shared glossaries, multilingual dictionaries, abbreviations, and emergency contacts.

## File Format
Each markdown file is heavily structured for RAG (Retrieval-Augmented Generation) optimization and includes:
- YAML Frontmatter (id, title, domain, category, keywords, aliases, intents, etc.)
- Multilingual Overview (English, Hindi, Hinglish)
- Quick Summary & Definitions
- Step-by-Step Protocols (Do's and Don'ts)
- Diagnostic Evaluation & Medical/Legal Management
- Government Initiatives & Schemes
- Frequently Asked Questions (FAQs)
- Common Myths vs Facts
- Conversation Examples for AI Training
- Search Optimization Keywords & Retrieval Tags

## Disclaimer
