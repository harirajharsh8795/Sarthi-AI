# Changelog

All notable changes to the Saarthi AI Knowledge Base will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-17

### Added
- **Medical Domain:** Comprehensive Multilingual Knowledge Base covering Anatomy, Cancer, Child Health, Chronic Diseases, Diagnostic Tests, First Aid, Heart, Infectious Diseases, Kidney, Liver, Medicines, Mental Health, Sexual Health, Symptoms, and Women's Health.
- **Legal Domain:** Comprehensive Multilingual Knowledge Base covering the Indian Constitution, Criminal Law (BNS), Cyber Crime, Sexual Offences, Family Law, Property Law, Consumer Law, Labour Law, and Police Procedures.
- **Banking Domain:** Comprehensive Multilingual Knowledge Base covering Banking Accounts, Cards, Loans, Investments, Cyber Security & Fraud, RBI policies, Payments, and Insurance.
- **Common Domain:** Added intent dictionaries, multilingual terminology glossaries, and emergency contacts.
- **Structure:** Implemented highly structured RAG-optimized Markdown templates with extensive YAML frontmatter.
- **Content Elements:** Added FAQs, Myths vs Facts, Conversation Examples, Do's and Don'ts, and Government Schemes to all domain files.
